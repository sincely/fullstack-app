# Auth

## POST 用户名+密码登录

POST /auth/login

> Body 请求参数

```json
{
    "userName": "Soybean",
    "password": "123456"
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|apifoxToken|header|string| 否 ||none|
|body|body|object| 否 ||none|
|» userName|body|string| 是 | 用户名|none|
|» password|body|string| 是 | 密码|none|

> 返回示例

> 200 Response

```json
{
  "data": {},
  "code": "string",
  "msg": "string"
}
```

|状态码|状态码含义|说明|数据模型|
